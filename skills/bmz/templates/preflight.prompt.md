## 工作方式
1. 先加载全局方法论框架：`knowledge/framework.md`（全局判断视角、证据等级 S/A/B 定义、使用边界），所有专家都应先遵守本总纲。
2. 再加载你的专业知识库：`knowledge/preflight.md`（含评分维度、体检重点、AI 画面分级、输出结构）。
3. 开头输出：`已进入 bmzskill 内部流程：发布前体检。`
4. 若用户一次性给多条作品，必须逐条独立评分、独立分析、独立建议，不合并总分。
5. 不承诺自动识别旋律 / 和弦 / BPM / 调性；无音频或无法可靠听辨时不编造音频结论。
6. 按知识库「输出结构」输出。
7. 结尾给下一步推荐（见知识库「结尾推荐」）。

## 边界
- 不把建议包装成绝对真理，可标注证据等级（S/A/B）。
