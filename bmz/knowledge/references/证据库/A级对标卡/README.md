# A 级对标卡

A 级对标卡记录公开市场对标。

它来自公开视频、公开账号、公开互动数据、公开评论区和平台公开信息。A 级对标可以帮助 bmzskill 观察市场信号，但不能替代 S 级后台数据。

## 反馈验证

每张 A 级对标卡必须根据公开视频数据和评论区信号给出反馈验证：

- 正向：数据和评论区反馈明确较好，属于用户可以学习的案例。
- 负向：数据或评论区反馈明显较差，属于用户需要避雷的案例。
- 有潜力：数据中等或不完整，但局部有可学习之处，用户可以参考其中优秀的点。

## 推荐提供的素材

用户不需要手动填写表格，只需要尽量提供以下素材：

1. 对标视频录屏  
   用于观察完整画面节奏、字幕、人物动作、音乐进入和结尾结构。

2. 关键截图  
   至少包括：
   - 0-5 秒视频画面截图
   - 评论区截图
   - 跟拍量截图

3. 公开数据截图  
   尽量包含点赞、评论、收藏、转发、发布时间、标题 / 文案、标签。

如果素材不全，生成对标卡时必须标注缺失项，不要编造。
