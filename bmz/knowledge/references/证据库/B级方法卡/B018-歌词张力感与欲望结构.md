# B018-歌词张力感与欲望结构

证据等级：B 级

适用模块：`/bmz-lyrics`、`/bmz-lyrics-check`、`/bmz-preflight`

## 核心判断

好歌词不只是“表达情绪”，而是让听众感到：这个人想要什么，但他不能直接得到。

歌词的性感、张力和活人感，通常来自这套结构：

```text
欲望 + 阻碍 + 伪装 + 禁忌 + 未完成 = 歌词张力
```

坏歌词常见的问题是：

```text
情绪 + 抽象词 + 解释 = 废话感
```

## 五个底层要素

### 1. 欲望

歌词里的人必须有清楚的想要：想被爱、想留下、想被选择、想解释、想体面离开、想重新开始、想证明自己没输、想让对方后悔。

如果歌词只有“我难过”，但没有“我想要什么”，就没有戏。

### 2. 阻碍

欲望必须被挡住，才会有张力：想说爱你但身份不合适，想挽留但对方已经不回头，想装洒脱但身体还记得，想忘记但生活处处提醒，想两清但关系里还有亏欠。

只写“我得不到”不够，必须让听众知道为什么得不到。

### 3. 伪装

真实的人很少直接说“我好痛”，更常见的是：没事，我早就忘了；照片我没删，只是懒得清；我不是放不下，是还没学会告别；其实我心动了，但赶路要紧，我没说。

伪装让歌词有第二层意思：表面一句话，背后藏着相反情绪。

### 4. 禁忌感

这里的禁忌不是猎奇，而是关系里的“不该”：不该想你但还想，不该回头但还是看了，不该心动但已经心动，不该嫉妒但忍不住，不该等但还在等，不该打扰但还想知道你近况。

“不该”会制造压抑，压抑会制造张力。

### 5. 未完成

好歌词不把故事讲完，而是停在一个还没解决的瞬间：话到嘴边没说，手伸出去又收回，门关上了但灯还亮着，消息打完又删了，人已经走了但位置还留着。

未完成感会让听众继续想。继续想，才会停留、收藏、评论。

## 好歌词与坏歌词的细节差别

### 情绪

好歌词的情绪有对象、有证据。坏歌词只有痛、累、等、想、放不下，但不知道谁让他这样，也不知道发生了什么。

### 逻辑

好歌词有起因、动作和结果。坏歌词经常是并列句堆叠：雨、星河、旧时光、孤独、命运轮流出现，但彼此没有因果。

### 文采

好歌词的文采是准：一个意象服务一个关系。坏歌词的文采是飘：漂亮词很多，但没有承担功能。

### 网感

好歌词前两句会快速筛选听众，让某类人立刻觉得“这首歌可能写我”。坏歌词前两句通常不告诉观众写给谁，短视频用户不会等它慢慢进入主题。

### 活人感

好歌词像真实的人在具体关系里说出来。坏歌词像在模拟悲伤，知道失恋歌该有雨、夜、等、回忆、沉默，但不知道真实的人在那个时刻会做什么。

## 审查时怎么用

审查一段歌词时，除了传播分和歌词分，还要额外判断“张力感”：

- 是否有明确欲望？
- 欲望是否被阻碍？
- 是否有表面说法和真实情绪之间的反差？
- 是否存在“不该但还是”的关系压抑？
- 是否停在一个未完成的瞬间？
- 是否有你我之间的电流，而不是单向倾诉？

如果这些都没有，即使句子看起来文艺，也只能判定为“有氛围但无张力”。

## 创作时怎么用

写歌词前，先确定这五件事：

```text
这个人想要什么？
他为什么不能直接得到？
他表面上在伪装什么？
这段关系里有什么“不该”？
这 8 句要停在哪个未完成的瞬间？
```

再写歌词。

不要先堆意象，也不要先解释情绪。

## 一句话总结

坏歌词让人觉得“这句好像挺 emo”。  
好歌词让人觉得“这句说的是我，而且它后面还有一整段关系”。
