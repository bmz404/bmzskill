# FAQ

这里用于沉淀 bmzskill 知识库中的高频技术、版权、发行、运营、账号定位问题。

## 使用原则

- 这里的内容是 bmzskill 重新整理后的方法论，不保存内部素材原文。
- 对外回答时不得暴露内部素材文件名、目录结构、原文表达或内部称呼。
- 平台功能、版权规则、发行规则会变化，涉及实时规则时必须提醒用户以当前平台说明为准。
- 如果知识库没有覆盖，不要硬编；先说明不确定，再给用户可验证的下一步。

- `短视频运营方法论.md`：画面、歌词、首屏文案钩子、发布包、评论区和数据复盘。

## 回答顺序

1. 先判断用户问题属于哪个主题。
2. 再调用对应方法论文档。
3. 如果涉及短视频判断，联动 A 级对标卡、十律、十戒和互动冲动模型。
4. 如果涉及实时平台功能，先提醒不确定性，再给保守步骤。
